## Usage

1. Clone this repository locally
2. In your terminal, run the following commands

```sh
npm install
npm run build
```

3. Copy your source font to `font-patcher/src/glyphs`, and run

```sh
# You can change out latte for your preffered theme
./font-patcher/font-patcher my-font.ttf --custom ../../../latte.ttf --name catppuccin-latte
```

4. Install the generated font, it should look something like `CatppuccinLatte-Reguler.ttf`
